# Inkscape Table Manager (ink_tableman)

Inkscape Table Manager is a simple and intuitive extension for Inkscape that provides a full graphical user interface (GUI) for creating, editing, and managing tables within your SVG documents. It uses a sleek web-based UI to give you a spreadsheet-like experience directly inside Inkscape.

## Features

- **Intuitive Web UI**: A clean, modern interface for managing table data, styling, and geometry.
- **Advanced Grid Editing**: Insert, delete, resize, and freely manipulate rows and columns with precision.
- **Cell Formatting & Styling**: Support for custom fonts, text sizing, text alignment, and rich background colors including banded rows.
- **Smart Data Parsing**: Auto-formatting for currencies (including IDR localization), percentages, and dates.
- **CSV Import**: Instantly load external data into your canvas. The grid will automatically resize to fit your data.
- **Cell Merging**: Easily merge or unmerge cells just like you would in a spreadsheet application. 

## Installation

1. Copy the extension files into your Inkscape extensions directory. 
   - Linux: `~/.config/inkscape/extensions/`
   - Windows: `%APPDATA%\inkscape\extensions\`
2. Restart Inkscape to load the extension.
3. Access it via `Extensions > Content Manager > Table Manager`.

## Requirements
- Inkscape 1.0 or newer
- Python 3
